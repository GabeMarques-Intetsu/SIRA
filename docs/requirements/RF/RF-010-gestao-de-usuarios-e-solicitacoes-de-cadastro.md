# RF-010 — Gestão de usuários e solicitações de cadastro

> **Tipo**: Requisito Funcional
> **Prioridade**: 🟠 Alta
> **Status**: 📝 Proposto

---

## Enunciado de negócio (sem termo técnico)

> **O sistema permite que o administrador cadastre, liste, busque, edite e exclua usuários, além de aprovar ou recusar solicitações de cadastro de professores.**

## Justificativa

CRUD de usuários pelo admin (cadastro direto, listagem com busca textual, edição de perfil, exclusão) somado ao fluxo de aprovação das solicitações de cadastro feitas via auto-serviço (EP-02). É o ponto de entrada efetivo de novos professores no sistema, por isso prioridade alta apesar de ser uma tela admin.
